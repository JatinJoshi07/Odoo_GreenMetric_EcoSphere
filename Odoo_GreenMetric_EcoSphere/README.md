# Odoo_GreenMetric_EcoSphere

A hackathon-ready EcoSphere ESG management platform starter built with React, Vite, Dexie, and an Express API foundation.

## Structure

- src/ - React frontend
- src/features/ - offline-first data layer
- server/ - Node.js API scaffold

## Run locally

- Frontend: npm run dev
- Backend: node server/server.js

## Notes

The current version focuses on an offline-first architecture shell with IndexedDB storage and a sync-friendly API scaffold.
